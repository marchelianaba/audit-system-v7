Konsep baru per catatan #6 & #7: alih-alih tab Setup / Output / Chat terpisah, v7 punya 2 workspace mega-tab yang masing-masing menyatukan semua aksi role tersebut.

2.1 Tab KKP — Workspace AT all-in-one

Layout single-page (scroll vertikal) dengan sections berurutan:

SectionAksi ATTool/Agen v7
① Konteks PenugasanLihat ST + KP (read-only) + Generate context.md via AI`build_context_md_template` + agen AT [MODE:CONTEXT]
② Upload DokumenDrag-drop dokumen objek → otomatis trigger digest pipeline`digest_generic` / `run_batch_*` per skill
③ Analisis AIKlik "Jalankan Analisis Agen AI" → progress streamAgen AT full run dgn skill-aware logic

⑤ Approval ATTombol "Setujui Final KKP" — render KKP.docx + lock`render_kkp_docx` (filter APPROVED) → status KKP_APPROVED

5. Roadmap Eksekusi Final (4 Sprint Revisi)

Sprint 1 — Visual Foundation + Workflow Restructure (week 1, ~18 jam)

  1. Rebrand INTEGRAL AI Workspace (2 jam): page titles, favicon, logo, footer
  2. Sidebar collapsible ungu + color palette (4 jam): Tailwind config + Layout component
  3. Toast + empty state (2 jam): sonner + ilustrasi roket
  4. Detail Penugasan layout (8 jam): grid 7 tahapan + Participants sidebar + Discussion + Activity Timeline
  5. Restructure menu (2 jam): rename Template Dokumen → Knowledge, Template Kriteria → CACM

Sprint 2 — Workspace AT + KT (week 2, ~20 jam)

  1. Tab KKP all-in-one (6 jam): konsolidasi tab Setup/Output/Chat existing → 5-section workspace AT
  2. Tab Konsep Laporan all-in-one (5 jam): workspace KT 5-section
  3. Template KP wiki + UI form PT (6 jam): 7 template + form picker + DOCX render
  4. Template PKP wiki + UI form KT (3 jam): auto-prefill dari KP

Sprint 3 — Workflow LRS + Survey + API (week 3, ~22 jam)

  1. Survey Pendahuluan conditional (3 jam): UI hanya muncul utk skill audit-*, ingestion + agen extension
  2. LRS LHP workflow (5 jam): DB model `LhpReview` + UI panel PT/PM
  3. Role PJ + finalisasi LHP (5 jam): DB enum extend + tombol Inspektur
  4. Daftar Penugasan DataTable SIMWAS-like (4 jam)
  5. API contract + endpoint v7 untuk SIMWAS (5 jam): I1+I2

Sprint 4 — Integrasi Live + UAT (week 4, ~15 jam)

  1. API endpoint terima ST dari SIMWAS (2 jam)
  2. Webhook bidirectional status sync (4 jam)
  3. UAT bersama tim SIMWAS (6 jam): 1 penugasan E2E via SIMWAS UI + v7 backend
  4. Dokumentasi handover + runbook ops (3 jam)

📊 Total: 4 sprint × ~18 jam/sprint ≈ 75 jam (~2 minggu kerja full-time)
